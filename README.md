# Git-Github-Commands


GIT != GITHUB

1. Check if Git is Installed
```bash 
   git --version
```


If Git is not installed, download it from 
```bash 
   git-scm.com.
```

2. Set Up Git (One-time Setup)
```bash
   git config --global user.name "Your Name"
   git config --global user.email "your-email@example.com"
```
